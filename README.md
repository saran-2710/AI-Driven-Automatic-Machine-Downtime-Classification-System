# AI-Driven-Automatic-Machine-Downtime-Classification-System
An Industry 4.0 prototype that uses machine learning to detect abnormal machine conditions and trigger automated safety response through a real-time monitoring dashboard and actuator control.

Overview:
   This project demonstrates a predictive maintenance system that monitors machine behavior and classifies downtime conditions using a machine learning model.
The system processes sensor data (simulated in the prototype), predicts machine health status, and automatically activates a physical actuator when abnormal conditions are detected.

The prototype integrates:
1.Machine Learning model (Random Forest)
2.Real-time monitoring dashboard
3.Embedded control using Arduino
4.Automated actuation mechanism

Key Features:
1.Real-time machine condition monitoring
2.Machine learning-based downtime classification
3.Failure probability visualization
4.Live dashboard interface
5.Automated actuator response for abnormal conditions
6.Expandable architecture for industrial sensor integration

Proof of Concept:
  The prototype demonstrates how machine learning predictions can trigger real-time mechanical responses.
  
When the system detects abnormal machine conditions:
 1.The ML model predicts failure probability.
 2.The dashboard displays the system status.
 3.The actuator is triggered automatically.
